# Student Report Card System 🎓

A beginner Python project that generates a report card for students.

## Features
- Stores student marks in a file
- Reads data from file
- Generates grade for each student based on marks

## Grading System
| Marks | Grade |
|-------|-------|
| 90+   | A     |
| 70-89 | B     |
| 50-69 | C     |
| 33-49 | D     |
| <33   | Fail  |

## Concepts Used
- Object Oriented Programming (OOPs)
- File I/O
- Loops
- Functions

## How to Run
1. Clone the repository
2. Run `python main.py`
